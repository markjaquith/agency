# @markjaquith/agency

Manages personal agents files

## Installation

```bash
bun add @markjaquith/agency
```

Or with npm:

```bash
npm install @markjaquith/agency
```

## Usage

```typescript
import "@markjaquith/agency"
```

## Requirements

- [Bun](https://bun.sh) >= 1.0.0 (recommended)
- TypeScript ^5

## Development

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```
